# AI Resume Analyzer

AI-powered Resume Analyzer built using:

- Python
- Streamlit
- Gemini API
- PyPDF

## Features

- Upload PDF Resume
- Extract Resume Text
- ATS Resume Score
- Strengths Analysis
- Weakness Detection
- Missing Skills Identification
- Improvement Suggestions

## Tech Stack

- Python
- Streamlit
- Gemini API
- PyPDF
- python-dotenv

## Run Locally

pip install -r requirements.txt

streamlit run resume_analyzer_app.py
